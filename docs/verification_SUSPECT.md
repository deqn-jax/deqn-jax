# Verification Report: DEQN-JAX ports vs. source notebooks

Cross-checks each ported model against the reference notebook it was derived from. Each section has three parts:

1. **Calibration match** — side-by-side table of constants.
2. **Ergodic residuals** — train the model, simulate a long trajectory under the trained policy, then compute per-equation residual statistics (mean / median / p99 / max of |residual|) averaged over MC shocks.
3. **Model-specific diagnostics** — closed-form error where available, constraint-binding check for the KKT model.

Residuals use the same per-sample formulation as the DEQN loss: `equations_fn(state, policy, next_state, next_policy)` averaged over 256 MC shock samples per state.

Generated by `scripts/verify_geneva_models.py` — run with `uv run python scripts/verify_geneva_models.py > docs/verification.md`.
